# Whiz POS Website Design Specification

This document outlines the design, structure, and content strategy for the official Whiz POS website. The goal is to provide a professional, modern, and user-friendly platform for users to download applications, view documentation, and understand the ecosystem.

## General Design Theme
*   **Style:** Modern, Minimalist, Clean, Professional.
*   **Color Palette:** Dark Mode preferred (Slate-900, Sky-400, Emerald-300) to match the application's "Glassmorphic" aesthetic.
*   **Typography:** 'Inter' for UI elements, bold headers.
*   **Navigation:** A sticky top navigation bar with the Logo (Whiz POS) on the left and links: Home, Windows POS, Mobile App, Back Office, Contact.

## 3. Mobile APK Page

**Goal:** Distribute the Android application and explain how to connect it to the Desktop Server.

### Header
*   **Title:** Whiz POS Mobile (Cashier App)
*   **Subtitle:** "Efficiency on the Go."

### Download Section
*   **Button:** "Download Android APK - v5.2.0"
*   **Requirements:** Android 8.0+, Camera permission (for QR scanning).

### Connection Guide (Crucial Section)
*   **Step 1:** Ensure your phone is on the same WiFi network as the Windows POS.
*   **Step 2:** Open Whiz POS Desktop -> Settings -> Mobile Sync to view the QR Code.
*   **Step 3:** Open the Mobile App and scan the QR Code to connect instantly.
*   **Visual:** Graphic showing a phone scanning a computer screen.

### Features
*   **Tableside Ordering:** Send orders directly to the kitchen printer.
*   **Secure Login:** PIN or QR Badge authentication.
*   **Sync:** Automatic background synchronization with the main server.
