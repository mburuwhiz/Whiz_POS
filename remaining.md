# Remaining Tasks

This document outlines the features and improvements that are planned for future development.

## Back-Office Enhancements
- [ ] **Dashboard:** Create a main dashboard with key metrics like daily sales, top-selling products, and sales trends.
- [ ] **Advanced Reports:** Implement date filtering, exporting to CSV/PDF, and more detailed transaction views.
- [ ] **Expense Management:** Add functionality to create, edit, and delete expenses from the back-office.
- [ ] **Credit Customer Management:** Allow viewing of individual credit sales and recording payments against them.
- [ ] **Business Details Management:** Add a page to update business information (name, address, etc.).

## POS Enhancements
- [ ] **Formal "Closing Day" Process:** Implement a dedicated "End of Day" workflow that archives the day's sales and resets the daily totals.
- [ ] **User Roles & Permissions:** Enforce more granular permissions for different user roles (e.g., restricting cashiers from accessing settings).
- [ ] **Inventory Tracking:** Improve stock tracking with features like purchase orders and supplier management.

## General
- [ ] **Automated Testing:** Implement a comprehensive test suite to improve code quality and prevent regressions.
